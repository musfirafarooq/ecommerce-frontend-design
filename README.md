# 📖 Project Overview

**NovaNode** is a fictional premium electronics and technology store created as the final submission for the **DevelopersHub Frontend Development Internship**.

The objective of this internship project was to transform the supplied **Figma desktop design** into a fully functional frontend application while following modern web development practices.

The website focuses on delivering a clean shopping experience by combining semantic HTML, modern CSS layouts, and interactive JavaScript functionality without relying on any frontend frameworks.

Every page has been designed to maintain a consistent visual identity while providing users with an intuitive browsing and shopping experience.

# 🎯 Project Objectives

The primary objectives of this project were:

* Convert the provided Figma design into pixel-accurate webpages.
* Develop reusable website components.
* Create a responsive desktop layout using modern CSS.
* Implement JavaScript-based interactivity.
* Practice semantic HTML structure.
* Build a maintainable and scalable frontend architecture.
* Demonstrate understanding of DOM manipulation.
* Implement client-side shopping cart functionality using LocalStorage.
* Maintain clean, organized, and documented source code.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/musfirafarooq/ecommerce-frontend-design.git
```

Navigate to the project:

```bash
cd ecommerce-frontend-design
```

Open **index.html** directly in your browser or use a local development server:

```bash
python -m http.server 8000
```

Open:

```
http://localhost:8000
```

---

# 🌍 Deployment

The project can be deployed using **GitHub Pages**.

1. Push the repository to GitHub.
2. Open **Settings → Pages**.
3. Select:

   * Branch: `main`
   * Folder: `/ (root)`
4. Save the settings.

The project will be available at:

```
https://musfirafarooq.github.io/ecommerce-frontend-design/
```
